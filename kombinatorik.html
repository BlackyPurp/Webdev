Willkommen zur Seite über Kombinatorik!

Die kombinatorik ist ein breit aufgebautes Feld der Mathematik welches sich mit der Berechnung von Anzahlen beschäftigt.


Du kannst dir Zur Erklärung der Grundlagen der Kombinatorik das Video von Daniel Jung anschauen.



1. Allgemeines Zählerprinzip

Definition!
Gegeben sind K mengen: m1...m2...m3k, welche jeweils n:n1...n2..nk elemente besitzen, dann lassen sich n1*n2*nk verschiedene k-Tupel zusammenstellen.

Ein Beispiel: Tim hat 5 T-Shirts, 3 Hosen und 2 paar socken.
Wie oft muss Tim sich umziehen, damit er mal alle möglichen kombinationen von Shirts,Hosen und Socken anhatte?

Er kann Jeweils jedes Tshirt 5*
jede Hose 5*3*
jedes paar socken 5*3*2
= 30 Mögliche Outfits, welche Tim anprobieren kann.

Übungen zum Allgemeinem Zählerprinzip

Wenn Tim sich jetzt 2 neue Hosen und noch 2 Verschiedene Jacken zum Anziehen für sein Outfit hinzufügt, wieviele Mögliche Kombinationen wären dann für ihn möglich?

2. Permutationen

Definition!
Eine Permutation ist eine K-Auswahl(spricht man wählt k mal) aus einer n-menge. Jeodch gibt es bei der Permutation zwei verschiedene arten. Einmal Eine Permutation mit Wiederholung und eine ohne.

Permutation mit Wiederholung

Eine Permutation mit Wiederholung ist eine anordnung von n objekte, von denen manche nicht unterscheidbar sind.
Formel hierfür ist:
n ! k 1 ! ⋅ k n !

Permutation ohne Wiederholung

Eine Permutation ohne Wiederholung ist eine Anordung von n Objekten, welche nicht gleich sind.
Formel Hiefür ist:
n !

Übungen Zu Permutationen


Tim hat 5 Karten mit verschiedenen Bildern auf der Hand, wie viele Möglichkeiten gibt es, die Karten in einer Reihe auf den Tisch zu legen?

3. Variationen

Definition!
k auswahl aus n-menge -> es wird eine Stichprobe betrachtet.
Reihenfolge der Elemente wird berücksichtigt -> "Geordnete Stichprobe"

Variation ohne Wiederholung

Bei einer Variation ohne Wiederholung werden k aus n Objekten unter beachtung der Reihenfolge ausgewählt, wobei jedes Objekt nur einmal ausgewählt werden kann.
Formel hierfür ist: n ! ( n − k ) !

Variationen mit Wiederholung

Bei einer Variaton mit Wiederholung werden k aus n Objekten under Beachtung der Reihenfolge ausgewählt, wobei Objekte auch mehrfach ausgewählt werden können. Formel hierfür ist:
n ⋅ n ⋅ … ⋅ n = n k

Übungen zur Variation


Auf einem Tisch liegen 10 verschiedene karte. Es sollen 4 Karten unter Beachtung der Reihenfolge und mit zurücklegen gezogen werden. Wie viele möglichkeiten gibt es?

4. Kombinationen

Definition! auswahl aus n-menge -> es wird eine Stichprobe betrachtet.
Reihenfolge der Elemente wird nicht berücksichtigt -> "Ungeordnete Stichprobe"

Kombinationen ohne Wiederholung

Bei einer Kombination ohne Wiederholung werden k aus n Objekten ohne Beachtung der Reihenfolge ausgewählt, wobei jedes Objekt nur einmal ausgewählt werden kann.Formel hierfür:
( n k )

Kombination mit Wiederholung

Bei einer Kombination mit Wiederholung werden k aus n Objekten ohne Beachtung der Reihenfolge ausgewählt, wobei Objekte auch mehrfrach ausgewählt werden können. Formel hierfür:
( n + k − 1 k )

Übungen zu Kombination


Auf einem Tisch liegen 10 verschiedene karte. Es sollen 4 Karten ohne Beachtung der Reihenfolge und ohne zurücklegen gezogen werden. Wie viele möglichkeiten gibt es?
