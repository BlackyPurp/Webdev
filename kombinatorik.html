Allgemeines Zaehlerprinzip

Definition!
Gegeben sind K mengen: m1...m2...m3k, welche jeweils n:n1...n2..nk elemente besitzen, dann lassen sich n1*n2*nk verschiedene k-Tupel zusammenstellen.

Ein Beispiel: Tim hat 5 T-Shirts, 3 Hosen und 2 paar socken.
Wie oft muss Tim sich umziehen, damit er mal alle moeglichen kombinationen von Shirts,Hosen und Socken anhatte?

Er kann Jeweils jedes Tshirt 5*
jede Hose 5*3*
jedes paar socken 5*3*2
= 30 Moegliche Outfits, welche Tim anprobieren kann.

Uebungen zum Allgemeinem Zaehlerprinzip

Wenn Tim sich jetzt 2 neue Hosen und noch 2 Verschiedene Jacken zum Anziehen fuer sein Outfit hinzufuegt, wieviele Moegliche Kombinationen waeren dann fuer ihn mogelich?

Permutationen

Definition!
Eine Permutation ist eine K-Auswahl(spricht man waehlt k mal) aus einer n-menge. Jeodch gibt es bei der Permutation zwei verschiedene arten. Einmal Eine Permutation mit Wiederholung und eine ohne.

Permutation mit Wiederholung

Eine Permutation mit Wiederholung ist eine anordnung von n objekte, von denen manche nicht unterscheidbar sind.
Formel hierfuer ist:
n ! k 1 ! ⋅ k 2 ! ⋅ … ⋅ k s !

Permutation ohne Wiederholung

Eine Permutation ohne Wiederholung ist eine Anordung von n Objekten, welche nicht gleich sind.
Formel Hiefuer ist:
n !

Uebungen Zu Permutationen


Tim hat 5 Karten mit verschiedenen Bildern auf der Hand, wie viele Moeglichkeiten gibt es, die Karten in einer Reihe auf den Tisch zu legen?


Variationen

Definition!
k auswahl aus n-menge -> es wird eine Stichprobe betrachtet.
Reihenfolge der Elemente wird beruecksichtigt -> "Geordnete Stichprobe"

Variation ohne Wiederholung

Bei einer Variation ohne Wiederholung werden k aus n Objekten unter beachtung der Reihenfolge ausgewaehlt, wobei jedes Objekt nur einmal ausgewaehlt werden kann.
Formel hierfuer ist: n ! ( n − k ) !

Variationen mit Wiederholung

Bei einer Variaton mit Wiederholung werden k aus n Objekten under Beachtung der Reihenfolge ausgewaehlt, wobei Objekte auch mehrfach ausgewaehlt werden koennen. Formel hierfuer ist:
n ⋅ n ⋅ … ⋅ n = n k

Uebungen zur Variation


Auf einem Tisch liegen 10 verschiedene karte. Es sollen 4 Karten unter Beachtung der Reihenfolge und mit zuruecklegen gezogen werden. Wie viele moeglichkeiten gibt es?

Kombinationen

Definition! auswahl aus n-menge -> es wird eine Stichprobe betrachtet.
Reihenfolge der Elemente wird nicht beruecksichtigt -> "Ungeordnete Stichprobe"

Kombinationen ohne Wiederholung

Bei einer Kombination ohne Wiederholung werden k aus n Objekten ohne Beachtung der Reihenfolge ausgewaehlt, wobei jedes Objekt nur einmal ausgewaehlt werden kann.Formel hierfuer:
( n k )

Kombination mit Wiederholung

Bei einer Kombination mit Wiederholung werden k aus n Objekten ohne Beachtung der Reihenfolge ausgewaehlt, wobei Objekte auch mehrfrach ausgewaehlt werden koennen. Formel hierfuer:
( n + k − 1 k )

Uebungen zu Kombination


Auf einem Tisch liegen 10 verschiedene karte. Es sollen 4 Karten ohne Beachtung der Reihenfolge und ohne zuruecklegen gezogen werden. Wie viele moeglichkeiten gibt es?
