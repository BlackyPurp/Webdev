Willkommen zur Seite ueber Absolute / Relative Haeufigkeit!

Die Haeufigkeit ist das Feld der Mathematik welches sich mit der Berechnung der Haeufigkeitsverteilung von bestimmten Ereignissen Beschaefftigt.


Du kannst dir Zur Erklaerung der Grundlagen der Haeufigkeit das Video von Daniel Jung anschauen.



1. Definition der Haeufigkeit

Definition!
Tritt ein Ereignis bei n-Versuchen k-mal ein, so heißt die Zahl: h n ( E ) = k n

2. Berechnen der Relativen Haeufigkeit

Definition!
Die relative Häufigkeit ist die absolute Häufigkeit dividiert durch die Anzahl der Versuche: Relative Häufigkeit = Absolute Häufigkeit Anzahl der Versuche
oder

h n ( E ) = H n ( E ) n

3. Eigenschaften

Die Relative Haeufigkeit besitzt 6 verschiedene Eigenschaften.

Die relative Häufigkeit nimmt Werte zwischen 0 und 1 an: 0 ≤ h n ( E ) ≤ 1


Die relative Häufigkeit des sicheren Ereignisses ist 1. h n ( Ω ) = 1


Die relative Häufigkeit des unmöglichen Ereignisses ist 0 h n ( { } ) = 0


Jedes Ereignis E und sein Gegenereignis E ― ergänzen sich zum Ergebnisraum Ω . h n ( E ) + h n ( E ― ) = 1


Die relative Häufigkeit des Ereignisses E entspricht der Summe der relativen Häufigkeiten der Ergebnisse w1, w2, ..., wk, aus denen das Ereignis E zusammengesetzt ist. h n ( E ) = ∑ i = 1 k h n ( { ω i } )


Sind die beiden Ereignisse A und B unvereinbar so lautet die formel: h n ( A ∪ B ) = h n ( A ) + h n ( B )

Sonst Gilt:
h n ( A ∪ B ) = h n ( A ) + h n ( B ) − h n ( A ∩ B )

4. Absolute Haeufigkeit Berechnen

Definition! Die absolute Häufigkeit lässt sich aus der relativen Häufigkeit und der Anzahl der Versuche berechnen A Häufigkeit = R Häufigkeit ⋅ Anzahl der Versuch H n ( E ) = h n ( E ) ⋅ n
